### Color Adjustments
```

- Background: White (#FFFFFF)

- Border: rgba(162,161,168,0.1)

Dark Mode:
- Background: Card (#1C1C24)
- Text: White (#FFFFFF)

- Primary: #7152F3 (Purple)
- Success: Green
- Warning: Amber

```
Light Mode Timeline Line:
  Purple → Purple (50%) → Gray (20%)

## 🔄 State Variations

### 1. High-Score Applicant (≥80)
```
✅ Green score badge
✅ All progress bars green
✅ "Schedule Interview" button (high priority)
✅ Top Candidate badge visible
✅ Strengths list populated
✅ "No significant risks identified"
```

### 2. Medium-Score Applicant (60-79)
```
⚠️ Yellow score badge
⚠️ Progress bars yellow/green mix
⚠️ Standard action buttons
⚠️ Top Candidate badge visible (≥70)
⚠️ Some risks may be listed
⚠️ Standard timeline
```

### 3. Low-Score Applicant (<60)
```
❌ Red score badge
❌ Progress bars red/yellow
❌ Standard action buttons
❌ No Top Candidate badge
❌ Risks likely listed
❌ Timeline may show incomplete docs
```

### 4. New Applicant (No AI Data Yet)
```
📝 Generate button shown
📝 "Get AI-powered insights..." message
📝 No score displayed
📝 Timeline shows basic events
📝 Click to generate analysis
```

### 5. Loading State
```
⏳ Spinner with sparkle icon
⏳ "Analyzing applicant profile..."
⏳ "This may take a few moments"
⏳ No other content shown
```

### 6. Error State
```
❌ Red border
❌ Alert icon
❌ Error message displayed
❌ "Try Again" button
❌ Click to retry generation
```

## 🎯 Visual Hierarchy

### Priority Levels
```
1. HIGHEST: AI Score Badge
   - Large, colored, top-right
   - First thing you see

2. HIGH: Summary Text
   - Purple-bordered box
   - Highlighted background

3. MEDIUM: Strengths & Risks
   - Icon headers
   - Bulleted lists

4. LOW: Tags
   - Small, bottom
   - Gray background
```
